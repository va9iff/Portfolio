Vibrant

Online playground and snippets for exploring Plada practically.

It has unique interface to test the library efficiently.
You can start the main loop, or stop by just clicking a button.
You can try snippets by clicking one.
There is a code editor to modify the snippet too.
Once you modified, it will remember your last file.
So you can work or test on a file without the fear of losing it.
Keep in mind Vibrant uses an old version on Plada.


Plada

Simulation and Visualisation library focused on syntax.

Currently Plada isn't published.
Vibrant project uses an old version of Plada, but it got really far from this and become sophisticated readable and modular library.
Its syntax is even suitable for kids start to learning programming and constructing logic, and functionality can go as deep as your imagination.
We are really working hard to make its gears stable, to keep its codes clean and to run well on every device.
